Cellular signaling Pathway Status Board
using Object-Oriented Programming in Java

A real-time status board for a cellular signaling pathway system, updating activities based on a virtual simulation of a cell, leveraging key object-oriented programming concepts such as polymorphism, inheritance, observer pattern, command pattern, and strategy pattern.



Background

While brainstorming project ideas that could effectively utilize key object-oriented design concepts such as inheritance, abstraction and composition, I thought that modeling the structure of molecules within a cell and the cross-talk between signaling pathways would be an excellent application. Various molecules, such as ligands, receptors, and transcription factors, can be grouped into categories using polymorphism and inheritance. Simulating pathways in a cell is akin to running a game, and I utilized the observer pattern to dynamically display real-time signal states on specific cards in the user interface.



Implementation

The simulation begins with extracellular ligands binding to their specific receptors on the cell surface, triggering the activation of translocation factors that move into the nucleus. Initially, molecules within the outer and inner cell are set according to their real-life ratios, while the nucleus starts empty. Regulatory dynamics, such as deactivation or dephosphorylation of molecules, are also incorporated to reflect realistic behavior.

  • Major Pathways in this project

    • - TGFB Pathway
    • - JAK-STAT Pathway
    • - Ras-MAP Pathway
    • - MAPK Pathway

    Each pathway typically progresses through phases that involve shared molecules. These pathways were structured using strategy pattern each commonly have phase 1, phase 2, phase 3 that are interchangeable.

  • Base structure

    • - Ligands directory: Cytokine, EGF, TGF-β classes
    • - Receptors directory: TGF- β receptors, RTK, EPOR classes
    • - Kinases directory : JAK2, MEK, Raf, MAPkinase classes
    • - Transcription Factors directory: Smad, Stat classes
    • - Regulatory Protein directory: SHP1, Ski, SOCS, SOS classes
    • - GDP, GNP, Ras and Tyrosine Residues
    • - Outer Cell class: Ligands exist
    • - Inner Cell class: Receptors, Kinases, Transcription factors, Regulatory protein, GDP, GNP,
          Ras, Tyrosine Residue exist
    • - Nucleus : Transcription factors enter as the simulation runs
    • - Simulation world class: simulating four pathways involving interactions from outer cell to inner cell and nucleus
    • I used factory pattern to generate each instance belonging to each group above.

  • Signaling interaction mechanism

    • - Bind Command class
    • - Phosphorylate Command class
    • - Dephosphorylate Command class
    • - Activate Command class
    • - Deactivate Command class

      • I implemented the Command Pattern to execute commands in various cell interaction scenarios, with the processes displayed in the UI using the Observer Pattern.

  • Observer UI mapping to commands

    • Through the Observer Pattern, different observer UI components are mapped to specific commands, ensuring that each observer receives updates relevant to its associated command.

  • Implemention

    • TGF β Pathway : TGF β binds to TGF β receptor if it's not bound yet (phase 1), and then the receptor activates the Smad, the transcription factor (phase 2), and smad complex enters the Nucleus (phase 3).
    • JAK-STAT pathway: Cytokine binds to its receptor EpoR, and receptor phoshporylates (phase 1), and the Jak2, the kinase, also binds to the receptor and phoshporylates the tyrosine residue. (phase 2). The transcription factor Stat binds to tyrosine residue (phase 3), and then the stat goes into nucleus(phase 4).
    • Ras-MAP Pathway : EGF binds to its receptor RTK (phase 1), and Ras becomes active as it binds to GDP while GDP exchanges to GTP (phase 2). When GTP hydrolyzes to GDP, it Ras becomes inactive again(phase three).
    • MAPK Pathway: When Ras sets active in Ras-MAP Pathway by binding to GTP, Raf binds to Ras-GTP , then MAP kinase becomes active (Phase 1). Raf unbinds from Ras-GTP and MEK, MEK phosphrylates Map Kinase(phase 2). MAP kinase enters nucleus(Phase three).
    • Regulation: In each pathway, regulatory dynamics are involved, where regulatory protein such as SHP1, Ski, SOCS, SOS deactivates receptors or complexes.

  • Simulation

    • Using Runnable java interface, I treated each pathway as a task that can be executed by a thread, so that multilple threads run concurrently, while maintaining the order of phase 1,2,3 in each pathway.
